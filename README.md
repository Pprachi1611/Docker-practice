## 📖 About

This repository contains my hands-on Docker practice projects created while learning **Docker**, **Docker Compose**, **container networking**, **multi-container applications**, and **AWS Elastic Container Registry (ECR)**.

The projects are organized in a progressive manner, starting from Docker fundamentals and gradually moving toward real-world application containerization and cloud deployment.

This repository will continue to grow as I explore more Docker and DevOps concepts.

---

## 🎯 Learning Objectives

- Learn Docker fundamentals
- Build Docker images
- Manage Docker containers
- Write Dockerfiles
- Create custom Docker networks
- Build multi-container applications
- Work with Docker Compose
- Deploy applications using containers
- Push Docker images to AWS ECR
- Prepare for DevOps and Cloud roles
