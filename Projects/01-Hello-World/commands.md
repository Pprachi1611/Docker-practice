# Commands Used

## Check Docker Version

```bash
docker --version
```

Displays the installed Docker version.

---

## Check Docker Information

```bash
docker info
```

Displays detailed information about the Docker Engine.

---

## Run Hello World

```bash
docker run hello-world
```

Downloads the image (if necessary), creates a container, and runs it.

---

## List Images

```bash
docker images
```

Lists all Docker images stored locally.

---

## List Running Containers

```bash
docker ps
```

Shows only running containers.

---

## List All Containers

```bash
docker ps -a
```

Shows both running and stopped containers.
