# Commands Used

## Install Dependencies

```bash
npm install
```

Installs all required Node.js packages.

---

## Start Application

```bash
npm start
```

Starts the Express server.

---

## Start in Development Mode

```bash
npm run dev
```

Runs the application using Nodemon.

---

# Docker Compose

## Build and Start Containers

```bash
docker compose -f docker-compose.yml up -d --build
```

Builds the application image (if required) and starts all services.

---

## Start Existing Containers

```bash
docker compose -f docker-compose.yml up -d
```

Starts all services without rebuilding the image.

---

## Stop and Remove Containers

```bash
docker compose -f docker-compose.yml down
```

Stops and removes all containers, networks, and resources created by Docker Compose.

---

## Stop and Remove Containers with Volumes

```bash
docker compose -f docker-compose.yml down -v
```

Removes containers along with their associated volumes.

---

# Docker Network

## Create Network

```bash
docker network create profile-network
```

Creates a custom Docker bridge network.

---

## List Networks

```bash
docker network ls
```

Displays all available Docker networks.

---

# Docker Image

## Build Image

```bash
docker build -t profile-app .
```

Builds a Docker image from the Dockerfile.

---

## List Images

```bash
docker images
```

Displays all Docker images.

---

## Remove Image

```bash
docker rmi profile-app
```

or

```bash
docker rmi <image-id>
```

Removes a Docker image.

---

# Docker Containers

## Run Container

```bash
docker run -d --name profile-app -p 3000:3000 profile-app
```

Runs the application container.

---

## List Running Containers

```bash
docker ps
```

Displays currently running containers.

---

## List All Containers

```bash
docker ps -a
```

Displays all containers, including stopped ones.

---

## View Containers Created from an Image

```bash
docker ps -a --filter ancestor=profile-app
```

Lists containers created from a specific image.

---

## Stop Container

```bash
docker stop <container-name>
```

Stops a running container.

---

## Remove Container

```bash
docker rm <container-name>
```

Removes a stopped container.

---

## View Container Logs

```bash
docker logs <container-name>
```

Displays logs generated by the container.

---

# Useful Commands

## Inspect Docker Network

```bash
docker network inspect profile-network
```

Displays detailed information about a Docker network.

---

## Inspect Container

```bash
docker inspect <container-name>
```

Displays detailed information about a container.

---

## Remove Unused Images

```bash
docker image prune -f
```

Removes dangling and unused images.

---

## Remove Unused Containers

```bash
docker container prune -f
```

Removes all stopped containers.
